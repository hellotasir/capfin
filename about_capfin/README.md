# CapFin

A project to showcase my skills in the Dart programming language and its associated frameworks.

### Key Features

- 1. User Management
- 2. Security
- 3. Financial Operations
- 4. Report Generation
- 5. Notification
- 6. Chat

### User Relation in System

Customer <-> System <-> Banker

### What bankers can do in this system or platform?

- 1. Can create customer accounts
- 2. Can delete customers if requested
- 3. Can view activities and operations but cannot control or modify them
- 4. Can view and download customer reports or customer records
- 5. Can chat with customers as Customer Support

### What customers can do in this system or platform?

- 1. Can request an account with a form
- 2. Can request account deletion with a form
- 3. Can perform transactions
- 4. Can view activity and other history
- 5. Can chat with bankers as a customer for help
- 6. Receive notifications on any activities and transactions
- 7. Can perform basic financial operations


### System Requirements 
- 1. Strong Security
- 2. Clean UI for better user experience
- 3. Quality (a platform that can not fail in any testing)


### My Opinion

I hope these features and requirements are sufficient to effectively showcase my backend and frontend development skills in this project using the Dart language and its associated frameworks.
